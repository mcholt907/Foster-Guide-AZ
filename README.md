# Foster Guide AZ

Foster Guide AZ is a multilingual resource platform for Arizona foster youth, focused on rights, case navigation, services, wellness, and transition planning.

## Quick Links

- Product app (primary): [`web/README.md`](web/README.md)
- Contributor standards: [`AGENTS.md`](AGENTS.md)
- Supporting backend reference: [`server/`](server)
- Prototype/reference UI: [`app/`](app)
- Content and maintenance scripts: [`scripts/`](scripts)

## Repository Layout

- `web/` — Next.js 16 production PWA.
- `server/` — Express + RAG backend reference code.
- `app/` — Vite-based prototype/reference frontend.
- `docs/` — reports and planning artifacts.
- `Context/` — product specs and research material.

For setup, run, and contribution details, start with [`web/README.md`](web/README.md) and [`AGENTS.md`](AGENTS.md).
